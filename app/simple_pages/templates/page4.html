Github is a code hosting platform that allows you to collaborate with others

There are many tasks you can accomplish with GitHub like creating a repository, creating branch etc.

When you just starting with Github, you want to first download it and install and configure after

It is recommended to be connected over SSH when cloning a repository

If you want to create a new branch, use this command: git branch [branch-name]

When you are ready to merge your lets say branch with main use: git merge command

Git pull command is used to pull info from remote branch on GitHUb

You can use git log command to display version history

Git clone [url] command is used for cloning a repo that is already in Github

One of the challenges of GitHub is navigating around branches and main

Never forget to commit and push your changes to reflect them in your local host and other envir.

If you want changes pass your locl changes onto a branch or main in Github, you want to pull request

After you pull request, you need to merge your pull request!
